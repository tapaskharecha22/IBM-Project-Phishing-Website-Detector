# IBM-Project-Phishing-Website-Detector

## 🛡️ PhishGuard – Phishing Website Detection System

### Description

PhishGuard is a cybersecurity-focused web application that detects phishing websites using Machine Learning. The system analyzes URLs submitted by users and classifies them as either **legitimate** or **phishing**, helping protect users from credential theft, financial fraud, and malicious attacks.

### Key Features

* 🔍 Phishing URL Detection
* 🤖 Machine Learning-Based Classification
* ⚡ Real-Time Analysis
* 🛡️ Enhanced Cybersecurity Awareness
* 🌐 User-Friendly Web Interface

### Technologies Used

* Python
* Flask
* Scikit-learn
* Machine Learning
* HTML/CSS

### Objective

To strengthen cybersecurity by identifying malicious websites before users interact with them, reducing the risk of phishing attacks and online fraud.

### Output

* ✅ Legitimate Website
* 🚨 Phishing Website Detected

**Cybersecurity Domain:** Web Security | Threat Detection | Machine Learning | Phishing Prevention
